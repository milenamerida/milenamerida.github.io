Databases

  • SELECT * FROM Customers;

    • Returns all records from a table named "Customers".

  • SELECT * FROM Customers
    WHERE Country='USA';

    • It is used to extract only those records that fulfill a specified condition, in this case it will select all customers from the USA.

  • SELECT COUNT(*)
    FROM Products;

    • Returns the number of rows that matches a specified criterion, in this case the rows that contain the data PRODUCTS.

  • INSERT INTO Customers (CustomerName, ContactName, Address, City, PostalCode, Country)
    VALUES ('Cardinal', 'Tom B. Erichsen', 'Skagen 21', 'Stavanger', '4006', 'Norway');

    • It is used to insert new records in a table, either by specifying both column names and the values to be inserted or insert values for all the columns which don't need to be specified.

  • UPDATE Customers
    SET ContactName = 'Alfred Schmidt', City= 'Frankfurt'
    WHERE CustomerID = 1;

    • It is used to modify the existing records in a table, in this case the first customer with a new contact person and a new city.

SQL databases are systems to manage data for queries and updates. Using it in my daily life would help me retrieve data efficiently and management. It helps with decision-making, data process, and supports development.
